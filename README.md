# IT Support Labs & Documentation

This repository documents hands-on IT support and cybersecurity foundation labs
completed as part of my CompTIA A+ studies and TryHackMe training.

## Focus Areas
- Windows troubleshooting
- Linux fundamentals
- Networking basics (TCP/IP, DNS, DHCP)
- Security fundamentals
- Documentation and problem-solving methodology

## Tools & Platforms
- Windows 10/11
- Ubuntu Linux
- VirtualBox
- TryHackMe

## Goal
To build real-world troubleshooting skills and prepare for an IT Support / Help Desk role,
with a long-term path toward Security Operations (SOC).
