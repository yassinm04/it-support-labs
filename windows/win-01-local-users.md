# Lab Title: Windows Local Users and Permissions

## Problem
Practice creating and managing local user accounts on a Windows system.

## Environment
- OS: Windows 10
- Tools: Computer Management, Local Users and Groups

## Steps Performed
1. Created a new local user account
2. Assigned user to appropriate groups
3. Tested permissions by logging in as the user
4. Verified access restrictions

## Result
Successfully created and validated a local user account with limited permissions.

## Skills Demonstrated
- Windows user management
- Permissions
- Troubleshooting methodology
